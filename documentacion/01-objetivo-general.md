# Objetivo General

## Descripción del Proyecto

Actualmente la empresa constructora administra sus obras mediante WhatsApp, llamadas telefónicas y planillas de papel.

El proyecto busca desarrollar una aplicación web que permita:

- Registrar obras.
- Registrar empleados.
- Asignar empleados a obras.
- Asignar tareas.
- Controlar estados de las tareas.
- Consultar avances.
